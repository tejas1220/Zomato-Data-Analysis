# 🍽️ Zomato Data Analysis using Python

## 📌 Project Overview

This project analyzes the Zomato restaurant dataset using Python to extract meaningful business insights. The analysis focuses on customer ordering behavior, restaurant ratings, average order values, and online vs. offline ordering trends through Exploratory Data Analysis (EDA) and data visualization.

---

## 🎯 Objectives

- Clean and preprocess the Zomato dataset.
- Analyze restaurant ratings and customer preferences.
- Identify top-performing restaurants based on ratings and votes.
- Compare online and offline ordering trends.
- Visualize business insights using charts and heatmaps.

---

## 📂 Dataset

The dataset contains information such as:

- Other restaurant-related attributes

## 📊 Data Analysis Performed

- Data Cleaning and Preprocessing
- Handling Missing Values
- Exploratory Data Analysis (EDA)
- Restaurant Rating Analysis
- Online vs Offline Order Analysis
- Average Order Cost Analysis
- Votes Analysis
- Correlation Analysis using Heatmap
- Data Visualization using Bar Charts, Count Plots, Heatmaps, and Histograms

---

## 📈 Key Findings

- Restaurants offering **online ordering generally received higher customer ratings** than offline-only restaurants.
- Offline orders showed comparatively **lower average ratings**.
- Identified the **top-performing restaurants** based on ratings and customer votes.
- Analyzed customer ordering behavior using various visualizations.
- Heatmap analysis helped identify relationships between important features.
